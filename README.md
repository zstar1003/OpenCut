## 本地开发

开发命令：

```bash
bun install                # 安装相关依赖
bun run dev                # 使用 Turbopack 启动 Next.js 开发服务器
bun run build              # 构建生产版本
bun run lint               # 运行 Biome Lint
bun run lint:fix           # 自动修复 Lint 问题
bun run format             # 使用 Biome 格式化代码
```
